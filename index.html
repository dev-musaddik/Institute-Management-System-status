Core System Features

Authentication & Roles

Secure JWT-based login with role-based access control for Admins, Teachers, and Students.

Student & Teacher Profiles

Complete CRUD operations for managing student and teacher information, including departments and semesters.

Daily Attendance Calendar

Visual calendar for tracking daily attendance, with automated holiday marking for weekends and special events.

Absence Escalation Workflow

Automated notifications for 1, 3, and 7-day absences, leading to reports and student status changes.

Powerful Admin Dashboard

Configure institute settings, manage holidays, assign teachers to courses, and review absence reports.

Modern & Responsive UI

Built with React and TailwindCSS for a seamless experience on any device, from desktop to mobile.
